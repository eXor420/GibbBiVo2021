Es gibt eine sogenannte Impressumspflicht welche eine dazu verpflichtet dass ein Impressum auf der Website vorhanden sein muss. Das Impressum dient dazu dass der Nutzer der Website erkennen kann mit wem er es zu tun hat oder wer hinter der Website steckt.
